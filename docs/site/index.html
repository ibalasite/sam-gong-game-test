⚡ Quick Start

Local Development

# 1. Clone and install
git clone https://github.com/your-org/sam-gong-game.git
cd sam-gong-game
npm install

# 2. Setup environment
cp .env.example .env
# Edit .env with your local values

# 3. Start infrastructure (PostgreSQL + Redis)
docker-compose up -d

# 4. Run migrations
npm run migrate

# 5. Start game server
npm run dev:server

# 6. Start client (in another terminal)
cd client && npm run dev
