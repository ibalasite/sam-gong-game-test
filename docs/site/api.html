API Reference — REST + WebSocket Protocol

1. API Overview

項目值
Base URL (REST)https://api.samgong.io/api/v1
Base URL (WebSocket)wss://ws.samgong.io
協議HTTPS（TLS 1.2+）/ WSS（WebSocket Secure）
內容類型application/json（REST）/ Colyseus Binary Protocol（WS）
OpenAPI Spechttps://api.samgong.io/api/docs（Swagger UI）
版本策略URL Path 版本控制（/api/v1/）

1.1 認證方式

所有需要認證的端點使用 Bearer JWT（RS256）：

Authorization: Bearer {access_token}
  • Access Token TTL：≤ 1 小時
  • 封號後 60s 內 Token 失效（Redis 黑名單機制）
  • 不使用 Cookie 認證（CSRF 防護）

1.2 Rate Limits

層次對象限制說明
L1a全域500 req/min/IPNGINX Ingress 全域限流
L1bPOST /auth/otp/send5 req/min/IPOTP 防暴力（Redis）
L2aPOST /auth/login30 req/min/IP登入防暴力（Redis）
L2bPOST /player/rescue-chip3 req/min/player救援籌碼防重複請求
L3WebSocket 訊息10 msg/s/playerColyseus onMessage 速率限制
L4Admin API100 req/min/admin管理後台操作

2. Authentication Endpoints

POST /auth/otp/send

發送 OTP 驗證碼（手機號碼驗證）。

// Request
{ "phone": "+886912345678" }

// Response 200
{ "message": "OTP sent", "expires_in": 300 }

// Response 429
{ "error": "rate_limit_exceeded", "retry_after_seconds": 60 }

POST /auth/otp/verify

驗證 OTP 並取得 Token（新用戶自動建立帳號）。

// Request
{ "phone": "+886912345678", "otp": "123456", "birth_year": 1990 }

// Response 200
{
  "access_token": "eyJhbGci...",
  "refresh_token": "rt_...",
  "expires_in": 3600,
  "is_new_user": false
}

POST /auth/oauth/google

Google OAuth 登入。

// Request
{ "id_token": "google_id_token_here" }

// Response 200 — same as OTP verify response

POST /auth/refresh

刷新 Access Token（使用 Refresh Token）。

// Request
{ "refresh_token": "rt_..." }

// Response 200
{
  "access_token": "eyJhbGci...",
  "expires_in": 3600
}

POST /auth/logout

登出並撤銷 Refresh Token。需 Bearer Token。

// Request
{ "refresh_token": "rt_..." }

// Response 200
{ "message": "logged out" }

3. Player Endpoints

GET /player/me

取得當前玩家資料。需 Bearer Token。

// Response 200
{
  "id": "uuid",
  "display_name": "玩家名稱",
  "chip_balance": 95000,
  "is_minor": false,
  "age_verified": true,
  "tutorial_completed": true,
  "daily_play_seconds": 3600,
  "show_in_leaderboard": true
}

PUT /player/settings

更新玩家設定。需 Bearer Token。

// Request
{
  "display_name": "新名字",
  "music_volume": 60,
  "sfx_volume": 80,
  "vibration": true,
  "show_in_leaderboard": false
}

// Response 200
{ "updated": true }

POST /player/daily-chip

領取每日贈籌（每日限 1 次）。需 Bearer Token。

// Response 200
{ "chips_received": 10000, "new_balance": 105000 }

// Response 400
{ "error": "already_claimed_today" }

POST /player/rescue-chip

請求救援籌碼（籌碼 < 入場門檻時）。需 Bearer Token。

// Response 200
{ "chips_received": 100000, "new_balance": 100000 }

// Response 400
{ "error": "rescue_unavailable", "reason": "chip_balance_sufficient" }

// Response 403
{ "error": "rescue_chip_claimed", "reason": "already_claimed_today" }

POST /player/ad-reward

廣告獎勵籌碼。需 Bearer Token。

// Request
{ "ad_unit_id": "interstitial_001", "reward_amount": 5000 }

// Response 200
{ "chips_received": 5000, "new_balance": 100000 }

// Response 400 — ad already watched / invalid ad_unit_id
// Response 401 — unauthorized
// Response 429 — too many ad rewards today

7. WebSocket Protocol

Client 透過 Colyseus SDK 連線至 wss://ws.samgong.io。

7.1 連線方式

import Colyseus from "colyseus.js";

const client = new Colyseus.Client("wss://ws.samgong.io");
const room = await client.joinOrCreate("sam_gong", {
  tier: "bronze",
  token: access_token,  // JWT Bearer Token
});

7.2 Client → Server 訊息（onMessage）

訊息類型PhasePayload說明
banker_betbanker-bet{ amount: number }莊家下注（min_bet ≤ amount ≤ max_bet）
player_callplayer-bet{}閒家跟注（扣除 banker_bet_amount）
player_foldplayer-bet{}閒家棄牌（不扣籌碼）
request_rescue_chipswaiting/settled{}請求救援籌碼
leave_room任何{}主動離開房間
chat_message任何{ content: string }發送聊天訊息（≤100 字元）

7.3 Server → Client 廣播事件

事件PhasePayload說明
myHanddealing{ cards: Card[] }私人訊息：玩家自己的 3 張手牌
showdown_revealshowdown{ hands: Record<player_id, Card[]> }廣播所有未 Fold 手牌
settlement_resultsettled{ settlements: Settlement[] }結算結果廣播
rescue_chipswaiting/settled{ chips_received: number, new_balance: number }救援籌碼成功
rescue_not_availablewaiting/settled{ reason: string }救援籌碼失敗
anti_addiction_warning任何{ daily_seconds: number, limit_seconds: number }防沉迷 80% 警告
anti_addiction_limit任何{ reason: 'daily_limit' | 'minor_curfew' }防沉迷強制下線
my_session_infojoining{ daily_seconds_remaining: number }加入房間後推送今日剩餘時間
force_disconnect任何{ reason: string, code: number }強制斷線（封號/多裝置）

7.4 WebSocket Close Codes

Code含義Client 行為
4000Room 解散（玩家不足）返回大廳
4001Token 無效 / 過期刷新 Token 後重連
4002籌碼不足入場門檻提示並引導至救援籌碼
4003帳號已封禁顯示封號訊息
4004防沉迷強制下線顯示防沉迷提示
4005多裝置登入踢出提示「已在其他裝置登入」
