Sam Gong (三公) — Local Deployment Manual

This guide walks you through setting up and running the Sam Gong multiplayer card game locally, from bare prerequisites to a fully functional Minikube deployment.

node --version     # v20.x.x

2. Clone and Setup

git clone https://github.com/your-org/sam-gong-game.git
cd sam-gong-game

Generate RSA keys for JWT authentication:

mkdir -p keys
openssl genrsa -out keys/private.pem 4096
openssl rsa -in keys/private.pem -pubout -out keys/public.pem
chmod 600 keys/private.pem
Security: Never commit the keys/ directory to git. It is already listed in .gitignore.

3. Install Dependencies

# Backend dependencies
npm install

# Frontend dependencies (Cocos Creator client)
cd client && npm install && cd ..

5. Local Dev: Docker Compose (PostgreSQL + Redis)

# Start infrastructure services
docker-compose up -d

# Verify services are running
docker-compose ps

# Expected output:
# sam-gong-postgres  Up  0.0.0.0:5432->5432/tcp
# sam-gong-redis     Up  0.0.0.0:6379->6379/tcp

6. Run Database Migrations

# Run all pending migrations
npm run migrate

# Verify schema
psql -h localhost -U postgres -d sam_gong -c "\dt"

7. Start Development Servers

# Terminal 1: Start game server (Colyseus + REST API)
npm run dev:server

# Terminal 2: Start client (Cocos Creator web preview)
cd client && npm run dev

# Terminal 3 (optional): Watch TypeScript compilation
npm run build:watch

Services will be available at:

  • Game Server (WebSocket): ws://localhost:2567
  • REST API: http://localhost:3000
  • Swagger UI: http://localhost:3000/api/docs
  • Colyseus Monitor: http://localhost:2567/colyseus (dev only)
  • Client (Web): http://localhost:7456

8. Run Tests

# Run all unit tests
npm test

# Run with coverage
npm run test:coverage

# Run specific test file
npx jest src/game/HandEvaluator.test.ts

# Integration tests (requires running Docker services)
npm run test:integration

# Watch mode
npm run test:watch

9. Minikube Kubernetes Deployment

# Start Minikube with sufficient resources
minikube start --cpus=4 --memory=8192 --disk-size=30g

# Enable required addons
minikube addons enable ingress
minikube addons enable metrics-server

# Build Docker images using Minikube's Docker daemon
eval $(minikube docker-env)
docker build -t sam-gong-server:local -f Dockerfile.server .
docker build -t sam-gong-client:local -f Dockerfile.client .

# Create namespace and secrets
kubectl create namespace sam-gong-local
kubectl create secret generic sam-gong-secrets \
  --from-env-file=.env.k8s \
  -n sam-gong-local

# Deploy to Minikube
kubectl apply -f infra/k8s/ -n sam-gong-local

# Wait for pods to be ready
kubectl rollout status deployment/colyseus-game-server -n sam-gong-local
kubectl rollout status deployment/rest-api -n sam-gong-local

# Get service URL
minikube service sam-gong-ingress --url -n sam-gong-local

11. Troubleshooting

Database Connection Failed

# Check PostgreSQL container
docker-compose logs postgres

# Connect directly
psql -h localhost -U postgres -d sam_gong

# Reset database
docker-compose down -v && docker-compose up -d
npm run migrate

Redis Connection Issues

# Check Redis
docker-compose logs redis
redis-cli -h localhost -a dev_redis_password ping
# Expected: PONG

JWT Key Issues

# Regenerate keys
rm keys/private.pem keys/public.pem
openssl genrsa -out keys/private.pem 4096
openssl rsa -in keys/private.pem -pubout -out keys/public.pem

Port Already in Use

# Kill process on port
lsof -ti:2567 | xargs kill -9
lsof -ti:3000 | xargs kill -9
