♠ 三公規則說明

三公（Sam Gong）是一種東南亞傳統撲克牌比大小遊戲。每位玩家持 3 張牌，將牌面數字相加後取個位數，以點數大者獲勝。

  • A = 1 點；2–9 = 面值；10 / J / Q / K = 10 點
  • 三張牌加總後取個位數（mod 10）為實際點數
  • 公牌：三張牌加總個位數為 0（即整十）為最大，例如 J + Q + K = 30 → 公牌
  • 點數相同時比花色：黑桃 > 紅心 > 方塊 > 梅花
  • 莊家（Banker）與所有閒家（Player）單獨比較，勝者通吃

技術架構概覽

Server-Authoritative 設計

所有牌局決策（洗牌、發牌、判勝負、結算）均在 Colyseus Server 執行，Client 僅負責渲染。

[Cocos Creator Client] ←→ WebSocket ←→ [Colyseus Server]
                                              ↕
                                       [SamGongRoom]
                                              ↕
                               [Game Logic（Pure Functions）]
                                         deck / evaluator / settlement

反作弊核心

使用 @filter decorator 確保未翻牌的牌面資料永遠不傳送至其他玩家的客戶端。
