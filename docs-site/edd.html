1. System Overview

架構模式

  • 模式: Server-Authoritative + Client-Rendering
  • 通訊: WebSocket（Colyseus Protocol）
  • 狀態同步: @colyseus/schema Differential State Sync
  • 牌局邏輯: 100% Server-side（Client 僅 Render）

高階架構（文字版）

[Cocos Creator Client] ←→ WebSocket ←→ [Colyseus Server]
                                              ↕
                                       [SamGongRoom]
                                              ↕
                               [Game Logic Module（Pure Functions）]
                                              ↕
                                        [SQLite（稽核日誌）]

2. 設計原則

#原則說明
1Server-Authoritative所有牌局決策均在 Server 執行，Client 無法偽造
2Pure Functionsdeck / evaluator / settlement 為純函數，無副作用，可獨立單元測試
3Schema Filtering反作弊核心，未翻牌的其他玩家牌面永遠不下發至客戶端
4Immutable State每次狀態更新產生新物件，不修改現有狀態
5Fail Fast非法狀態轉換直接拋出錯誤，拒絕處理

4. 評分函數（核心邏輯）

// evaluator.ts
export function getCardPoint(rank: Rank): number {
  if (rank === 'A') return 1;
  if (['J', 'Q', 'K'].includes(rank)) return 10;
  return parseInt(rank, 10);
}

export function calculatePoints(cards: Card[]): number {
  const total = cards.reduce((sum, c) => sum + getCardPoint(c.rank), 0);
  const mod = total % 10;
  return mod === 0 ? 10 : mod; // 10 = 公牌
}

export function isGong(cards: Card[]): boolean {
  return calculatePoints(cards) === 10;
}

5. 反作弊：Schema Filtering

@filter decorator 機制

// 反作弊核心：opponent cards filtered until reveal phase
export class Card extends Schema {
  @filter(function(this: Card, client: Client, value, root: SamGongState) {
    const player = root.players.get(client.sessionId);
    if (!player) return false;
    // 翻牌前，只有自己的牌可見
    return root.phase === "reveal" ||
           root.phase === "settling" ||
           root.phase === "round_end" ||
           this.ownerId === client.sessionId;
  })
  @type("string") suit: Suit = "spades";

  @filter(/* same logic */)
  @type("string") rank: Rank = "A";
}

未翻牌時，其他玩家的 suit 和 rank 欄位不會包含在 State Diff 中，即使透過 DevTools 攔截也無法取得牌面資訊。

6. Rate Limiter

// 每個 session 每秒最多 10 次操作
const rateLimiter = new Map<string, number[]>();

function checkRateLimit(sessionId: string): boolean {
  const now = Date.now();
  const timestamps = rateLimiter.get(sessionId) || [];
  const recent = timestamps.filter(t => now - t < 1000);
  if (recent.length >= 10) return false; // 超過速率限制
  recent.push(now);
  rateLimiter.set(sessionId, recent);
  return true;
}
